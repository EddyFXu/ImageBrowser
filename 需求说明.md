# 小美图片查看器 (XiaoMei Image Viewer) - 需求规格说明书

## 1. 项目概述
小美图片查看器是一款专为 Windows 10/11 系统打造的轻量级、高性能图片浏览工具。项目核心目标是提供极致的视觉美感（梅兰竹菊主题）与流畅的操作体验，支持从基础的文件浏览到沉浸式的幻灯片播放。

---

## 2. 核心显示模式

### 2.1 资源浏览模式 (Browse Mode)
*   **布局结构**：
    *   **左侧**：标准系统目录树，支持逐层展开与点击定位。
    *   **右侧**：资源展示区。支持“详细列表”与“缩略图”两种视图切换。
*   **交互特性**：
    *   **平滑缩放**：支持通过 `Ctrl + 滚轮` 实现列表到缩略图大小的无缝切换。
    *   **快速进入**：双击图片文件直接进入“图片查看模式”；双击文件夹进入该目录。
    *   **自动换行**：缩略图视图下，图标根据窗口宽度自动排版。

### 2.2 图片查看模式 (View Mode)
*   **布局结构**：
    *   **左侧**：保留目录树，方便快速跳转目录。
    *   **右侧上方**：主展示区，占据大部分空间，用于显示单张大图。
    *   **右侧下方**：
        *   **缩略图预览行**：展示当前目录下所有图片，支持横向滚动。
        *   **属性信息栏**：显示分辨率、大小、格式、拍摄时间等 EXIF 信息。
*   **交互特性**：
    *   **智能缩放**：图片加载时自动调整比例以适应窗口。支持以鼠标位置为中心的滚轮缩放。
    *   **导航切换**：支持点击预览行、左右方向键、或浮动翻页按钮切换图片。
    *   **模式返回**：双击图片显示区域返回“资源浏览模式”。
    *   **中间定位**：切换图片时，下方预览行自动将当前选中项滚动至视觉中心。
    *   **图片旋转**：提供顺时针/逆时针 90 度旋转按钮，旋转后图片自动重新自适应窗口大小。
    *   **一键删除**：在浏览、查看、幻灯片模式下，均支持通过 `Delete` 键永久删除当前选中的文件或图片（带二次确认）。

---

### 2.3 幻灯片模式 (Slide Mode)
*   **全屏体验**：默认全屏展示，隐藏非必要 UI 元素，仅保留悬浮控制条。
*   **播放控制**：
    *   支持空格键暂停/播放，`Esc` 退出。
    *   支持顺序播放、随机播放、循环播放设置。
    *   可自定义播放间隔时间。

---

## 3. UI 与视觉规范

### 3.1 四君子主题系统
*   **梅 (Mei)**：冬日傲雪，简约白底，高雅清新（默认）。
*   **兰 (Lan)**：空谷幽兰，淡雅青色，静谧舒适。
*   **竹 (Zhu)**：翠竹劲节，自然绿色，护眼健康。
*   **菊 (Ju)**：金秋淡泊，深邃黑色，高级暗色模式。
*   **视觉增强**：各主题配有专属水墨风背景水印，提升文化底蕴。

### 3.2 界面交互细节
*   **磨砂玻璃效果**：顶层工具栏与浮动按钮采用极简磨砂风格。
*   **无边框设计**：自定义标题栏，支持双击最大化、拖拽移动。
*   **动态图标**：全矢量图标，支持高 DPI 缩放，状态切换时图标动态更新。

---

## 4. 功能特性与技术指标

### 4.1 数据管理
*   **排序功能**：支持按名称、大小、日期、类型进行递增/递减排序。
*   **状态持久化**：自动记忆窗口尺寸、位置、主题、上次访问路径及幻灯片偏好。
*   **落盘索引**：配置文件存储于 `%AppData%/ImageBrowser`，确保绿色无污染。

### 4.2 性能优化
*   **异步加载**：图片加载与缩略图生成在后台线程执行，确保 UI 响应。
*   **内存管理**：及时释放文件锁与位图资源，避免长久运行导致的内存溢出。
*   **自适应布局**：窗口尺寸变化时，图片缩放比例与布局元素实时重绘。
