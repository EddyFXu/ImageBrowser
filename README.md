# 小美图片查看器 (XiaoMei Image Viewer) v1.2.2

[![Platform](https://img.shields.io/badge/Platform-Windows%2010%2B-blue.svg)](https://www.microsoft.com/windows)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

小美图片查看器是一款专为提供极致视觉体验而设计的轻量级 Windows 图片浏览工具。它融合了中国传统的“梅兰竹菊”四君子文化与现代磨砂玻璃设计语言，在保证高性能大图加载的同时，提供了流畅的交互手感。

---

## 🚀 核心特性

### 🖼️ 多维浏览模式
- **资源浏览 (Browse)**：深度集成系统目录树，支持缩略图与详细列表平滑缩放。
- **图片查看 (View)**：沉浸式大图展示，带智能自适应缩放与缩略图导航行。
- **幻灯片 (Slide)**：全屏自动播放，支持随机/顺序切换及速度自定义。

### 🎨 东方美学 UI
- **四君子主题**：梅（雪白）、兰（淡青）、竹（翠绿）、菊（墨黑），各具韵味。
- **磨砂设计**：采用极简磨砂风格工具栏与悬浮控制，界面通透灵动。
- **动态交互**：无边框窗口设计，支持高 DPI 适配与图标状态感应。

### ⚡ 高性能架构
- **秒级加载**：优化大图解码与异步缩略图生成算法。
- **精准导航**：支持预览行“中心定位”逻辑，切换图片始终保持视觉焦点。
- **零污染设计**：设置自动持久化至 AppData，支持窗口状态与路径记忆。

## 🛠️ 编译与运行

项目采用纯 C# 与 WPF 开发，无需复杂环境即可通过 .NET Framework 4.0 编译器直接构建。

```powershell
# 推荐编译命令 (包含图标嵌入)
csc.exe /nologo /optimize /target:winexe /win32icon:"app.ico" /out:ImageBrowser.exe `
    /r:"C:\Windows\Microsoft.NET\Framework64\v4.0.30319\WPF\PresentationCore.dll" `
    /r:"C:\Windows\Microsoft.NET\Framework64\v4.0.30319\WPF\PresentationFramework.dll" `
    /r:"C:\Windows\Microsoft.NET\Framework64\v4.0.30319\WPF\WindowsBase.dll" `

    "App.cs" "MainWindow.cs" "MainViewModel.cs" "Models.cs" "ThemeVisuals.cs" "ThemeManager.cs" "Converters.cs" "Infrastructure.cs"
```

## 📅 更新日志

### v1.2.2 (2026-02-25)
- **新增功能**：
  - **文件删除支持**：在所有模式下均可通过 `Delete` 键删除当前选中的文件或图片，删除后自动切换至下一项。
- **交互优化**：
  - **删除确认**：增加二次确认弹窗，防止误删。

### v1.2.1 (2026-02-25)
- **新增功能**：
  - **图片旋转**：在查看模式工具栏新增顺时针/逆时针 90 度旋转功能。
- **修复与优化**：
  - **缩放算法重构**：引入“旋转感知”计算逻辑，完美解决旋转后或特定比例下图片溢出窗口出现滚动条的问题。
  - **自适应增强**：采用 99.9% 比例微调与强制布局刷新，确保图片在任何分辨率下均严丝合缝地契合窗口。

### v1.1.1 (2026-02-25)
- **功能增强**：
  - **多维排序**：支持按名称、大小、日期、类型进行递增/递减排序，表头带状态指示。
  - **排序继承**：图片查看模式与预览行自动继承并应用浏览模式的排序规则。
- **修复与优化**：
  - **自适应修复**：修复了因模式切换导致的图片无法自动适应窗口大小的问题。
  - **响应优化**：增加了窗口尺寸变化（SizeChanged）时的实时图片比例重算。
  - **文档重构**：全面优化了 README 与需求说明书的结构与内容。

### v1.1.0 (2026-02-24)
- **UI 改进**：预览行滚动按钮改为竖长条，实现中心定位逻辑。
- **稳定性**：修复双击卡死、目录树同步异常及 EXE 图标丢失问题。
