# 小美图片查看器 (XiaoMei Image Viewer) v1.0.9

## 简介
小美图片查看器是一款轻量级、高性能的 Windows 图片浏览工具，专为提供极致的视觉体验和流畅的操作手感而设计。它支持多种查看模式，拥有精美的界面主题，并针对大图查看进行了深度优化。

## 主要特性

### 1. 多种浏览模式
- **资源浏览模式**：左侧目录树，右侧文件列表/缩略图，支持无缝切换。双击图片直接进入查看模式。
- **图片查看模式**：专注于单张图片的大图查看，支持缩放、拖拽。双击图片区域返回浏览模式。
- **幻灯片模式**：全屏自动播放，沉浸式体验。

### 2. 极致的视觉体验
- **四君子主题**：
  - **梅 (Mei)**：冬日傲雪，简约高雅（默认主题）。
  - **兰 (Lan)**：空谷幽兰，淡雅清新。
  - **竹 (Zhu)**：翠竹劲节，护眼舒适。
  - **菊 (Ju)**：金秋淡泊，深邃高贵（暗色模式）。
- **水墨风水印**：每个主题配有专属的梅兰竹菊水墨背景水印，提升文化韵味。
- **现代 UI 设计**：
  - **无边框窗口**：自定义标题栏，完美融合系统边框。
  - **矢量图标**：全套矢量图标绘制，高分屏下依然清晰锐利。
  - **自适应滚动条**：滚动条样式随主题自动切换，深色/浅色模式完美适配。

### 3. 高性能与智能
- **极速加载**：优化大图加载算法，秒级打开高分辨率图片。
- **智能导航**：
  - 支持键盘方向键切换图片。
  - 幻灯片模式支持空格暂停/播放，ESC 退出。
  - 记忆播放设置（循环/随机、间隔时间）。
- **设置持久化**：自动保存窗口大小、位置、上次访问路径及幻灯片设置。
